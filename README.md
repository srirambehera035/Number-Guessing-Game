# Number Guessing Game

## Project Description

The Number Guessing Game is a console-based Java application that challenges players to guess a randomly generated number within a limited number of attempts. The game provides interactive hints after each guess, helping players determine whether the secret number is higher or lower than their current guess.

Designed using core Java concepts such as loops, conditional statements, methods, random number generation, and user input handling, this project offers an engaging way to practice problem-solving and logical thinking while demonstrating fundamental programming skills.

The game supports multiple rounds, maintains a cumulative score, and assigns a player rank based on overall performance. It also includes robust input validation to ensure a smooth and user-friendly experience.

## ✨ Features
- 🎲 Random number generation between 1 and 100
- 🔄 Multiple game rounds
- 🎯 Maximum of 10 attempts per round
- 💡 Intelligent hints:
  * "Much Higher" / "A Bit Higher"
  * "Much Lower" / "A Bit Lower"
- 🏆 Dynamic scoring system based on remaining attempts
- 📊 Total score tracking across rounds
- 🥇 Player ranking system:
  * Beginner
  * Intermediate
  * Expert
  * Grand Master
- ✅ Input validation for incorrect entries
- 🎨 Attractive console interface with banners and round summaries

## 🛠️ Technologies Used

- Object-Oriented Programming Concepts
- Java Scanner Class
- Java Random Class

## 🎮 How the Game Works
- The program generates a random number between 1 and 100.
- The player has 10 attempts to guess the number.
- After each guess, the game provides a hint indicating whether the secret number is higher or lower.
- The closer and faster the player guesses correctly, the higher the score earned.
- At the end of each round, the player can choose to play again.
- When the player exits, the game displays:
  - Total rounds played
  - Final score
  - Performance rank
